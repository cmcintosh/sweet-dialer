## Format-Specific Adjustments

### LinkedIn Posts (1350x1080)
- Use **horizontal compositions** with subjects centered
- Leave **top/bottom margin** for text overlays
- Good for: Team photos, office scenes, product shots

### Blog Featured (1920x1080)
- **Wide cinematic** compositions work best
- Use **bokeh backgrounds** for depth
- Good for: Hero images, landscape office views, abstract concepts

### Blog Thumbnails (1200x630)
- **Center-weighted** compositions
- Consider **square-ish framing**
- Good for: Icons, single subjects, clean product shots

## Prompt Building Formula

```
[SUBJECT] + [SETTING] + [LIGHTING] + [STYLE] + [TECHNICAL]

Example:
professional developer [SUBJECT] 
coding at standing desk in modern loft office [SETTING]
natural afternoon light through large windows [LIGHTING]
corporate tech photography style [STYLE]
photorealistic, sharp focus, 8k resolution [TECHNICAL]
```

## Recommended Negative Prompts

### Standard Negative (always include):
```
low quality, blurry, distorted, watermark, text, signature, cropped, out of frame, worst quality, low resolution, amateur photography, oversaturated, bad anatomy, disfigured, deformed, extra limbs, poorly drawn face, duplicate, morbid, mutilated, bad hands
```

### Extra (for portraits):
```
mutated hands, fused fingers, too many fingers, crossed eyes, asymmetrical face, bad teeth, wrinkles, acne
```

### Extra (for architecture/office):
```
distorted perspective, warped walls, curved lines, fisheye, lens flare, chromatic aberration
```

## Testing Checklist

When testing new prompts:

- [ ] Generate 4-8 variations with different seeds
- [ ] Check faces for distortion (enable FaceDetailer if needed)
- [ ] Verify text elements are NOT generated (use image editor for text)
- [ ] Confirm resolution matches target platform
- [ ] Review for unwanted watermarks or artifacts
- [ ] Test prompt with different lighting keywords (golden hour, soft, dramatic)
