# Sweet-Dialer

Twilio AI-Powered Dialer for SuiteCRM

## Overview

Sweet-Dialer is a comprehensive SuiteCRM module that integrates Twilio's Voice API to enable browser-based calling directly from your CRM.

## Features

- **Click-to-Call**: Initiate calls directly from Contact, Lead, and Account records
- **Inbound Call Routing**: Automatic routing to assigned agents
- **Browser-Based Dialer**: WebRTC-powered calling
- **Voicemail Management**: Custom voicemail with recording and playback
- **Call Recording**: Automatic recording with secure playback
- **Call Tracking**: Complete call history linked to CRM records
- **Multi-Agent Support**: Multiple CTI configurations per agent
- **Encryption**: AES-256 encryption for credentials

## Installation

1. Upload via SuiteCRM Module Loader
2. Configure Twilio credentials in Admin → Twilio Settings
3. Assign agent phone numbers to users

## Requirements

- SuiteCRM 7.12+ or 8.x
- PHP 7.4+ or PHP 8.0+
- MySQL 5.7+ or MariaDB 10.3+
- Twilio Account with Voice capabilities
